견체공학 챗봇 (강아지용품 전문)

[목표]
고객이 제품(어부바가방, 견글라스)을 이해하고, 빠르게 정보를 얻거나 상담할 수 있도록 돕는 챗봇을 구축합니다.
또한, 강아지 관련 정보를 제공하고 브랜드(견체공학)를 소개하는 역할도 수행합니다.

[초기 화면 구성]

메인 메시지: "무엇을 도와드릴까요?"

버튼 4개 배치:

어부바가방

견글라스

강아지 정보 톡톡

견체공학 소개

※ 버튼 클릭 시 해당 카테고리에 맞는 세부 질문 리스트 또는 답변 흐름으로 이동합니다.

[챗봇 답변 데이터셋]

제공된 CSV 파일 (견체공학_챗봇_보강본.csv)을 사용합니다.

각 질문에 대해

사용자 질문 예시를 참고하여 매칭

챗봇 답변을 그대로 출력

모든 답변 말미에 "더 도움이 필요하면 상담원 연결을 요청해주세요." 문구 추가

[대화 흐름 규칙]

사용자가 명확한 버튼을 누르면 해당 카테고리로 이동

버튼을 누르지 않고 질문을 입력하는 경우, 가장 유사한 소분류를 찾아 답변

답변 이후 "추가로 궁금한 점이 있으신가요?" 메시지와 버튼을 다시 출력

상담원 연결이 필요한 경우, "상담원 연결하기" 버튼 활성화

[기타 설정]

사용자 질문 오류 대응 문구:
→ "죄송합니다, 이해하지 못했어요. 선택지를 다시 안내드릴게요!"

대기 시간 5초 이상 무응답 시:
→ "궁금한 점이 있으시면 언제든 말씀해주세요! 😊" 자동 발화

[디자인 스타일 참고사항]

메인 테마: 따뜻하고 친근한 느낌 (베이지 톤, 강아지 일러스트 활용)

버튼 스타일: 라운드 모서리, 심플하고 깔끔한 디자인

폰트: 읽기 쉬운 고딕체 (Noto Sans, Spoqa Han Sans 등 추천)

[요약]

버튼 4개

CSV 데이터 기반 답변

간단명료한 사용자 흐름

부드러운 오류 대응 및 추가 질문 유도

플로우 디자인할 때 주의할 점

사용자가  2~3번 이내 클릭 으로 원하는 답을 얻을 수 있게 해야 합니다.

매 클릭 후 항상 다음 행동을 제안 해야 합니다. ("추가로 도와드릴까요?" 같은 문구)

너무 많은 질문 리스트를 한 번에 보여주지 말고, 카테고리별로 소화시키는 것이 좋습니다
